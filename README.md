# Portfolio

A personal portfolio and blog website built with React and Node.js.

- https://joaorihan.vercel.app

## Project Structure

- `frontend/`: React application
- `backend/`: Node.js server

## Getting Started

### Frontend

```bash
cd frontend
npm install
npm start
```

### Backend

```bash
cd backend
npm install
npm start
```

## Features

- Responsive design
- Contact form
- Project showcase
- About me section
- Blog functionality (soon!)

## Technologies Used

- React
- Node.js
- Express
- MongoDB
- Bootstrap
